** Usage
   The module may be used as follows:
#+BEGIN_EXAMPLE scheme
#!/usr/bin/guile \
-e main
!#

(use-modules (ice-9 pretty-print)
             (srfi srfi-41) ; Streams
             (ics))         ; Guile-ICS

(define (main args)
  (stream-for-each (lambda (ical-object)
                     (display "RESULT: \n")
                     (pretty-print ical-object))
                   (ics->stream)))
#+END_EXAMPLE
